## ⚡ START HERE

### Step 1: Open Cursor
```
Open your project folder in Cursor
Make sure it sees: index.html, app.js, style.css
```

### Step 2: First Feature - Login Modal
Go to your `CURSOR_GUIDE.md` file and copy the first prompt under **Feature 1**.

**Prompt 1: Create Login Modal HTML**
```
I have a Twitter clone (index.html, app.js, style.css). 
I need to add a login modal that:
1. Shows on page load if user is not logged in
2. Has input fields for username and handle
3. Has a "Sign In" button
4. Closes when user clicks Sign In
5. Should not interfere with existing HTML structure

Add the modal HTML to index.html (as the first element in body). 
Keep styling consistent with the existing design. 
Show me the updated index.html with the modal included.
```

✅ **What to do:**
1. Paste this into Cursor chat
2. Review the code Cursor generates
3. Accept the changes (copy the updated index.html to your file)
4. Test in browser - you should see a modal

---

### Step 3: Continue with Feature 1
Once HTML works, go back to `CURSOR_GUIDE.md` and use:
- **Prompt 2** for CSS styling
- **Prompt 3** for JavaScript functionality

Test after each step!

---

### Step 4: Feature 2 - Explore Modal
Once Feature 1 is complete, repeat the same process with Feature 2 prompts.

## 🆘 If Something Goes Wrong

**Cursor suggests breaking your existing code?**
→ Ask Cursor: "How do I integrate this safely without breaking my existing tweets and dark mode?"

**Modal doesn't appear?**
→ Check browser console (F12) for errors and show Cursor the error

**Can't figure out what Cursor did?**
→ Ask: "Can you explain what this code does in simple terms?"
